## Таблицы БД

- `markets` — метаданные рынков
- `orderbook` — снимки стакана
- `trades` — сделки
- `fee_rates` — комиссии по токенам
- `features` — рассчитанные фичи
- `signals` — сигналы модели
- `orders` — наши ордера
- `results` — результаты бэктеста

## Валидация модели

- TimeSeriesSplit / walk-forward
- Запрет утечки: train до t, test на t+1..t+k
